# Site Web HERMES - Surveillance de la Qualité de l'Air

## Description
Site web interactif pour visualiser les données de pollution collectées par le drone HERMES.

## Fonctionnalités
- 📊 Graphique interactif avec Chart.js
- 🎯 Sélection des polluants à afficher
- 📈 Statistiques moyennes en temps réel
- 🎨 Design moderne inspiré des drones et de la technologie
- 📱 Interface responsive

## Utilisation
1. Ouvrez `index.html` dans un navigateur web moderne
2. Le site charge automatiquement les données du fichier `mesure_20260310_001451.txt`
3. Utilisez les cases à cocher pour sélectionner les polluants à visualiser
4. Par défaut, tous les polluants sont affichés

## Polluants surveillés
- **CO** : Monoxyde de Carbone (ppm)
- **NH₃** : Ammoniac (ppm)
- **CH₄** : Méthane (ppm)
- **PM2.5** : Particules fines (μg/m³)
- **PM10** : Particules (μg/m³)

## Technologies utilisées
- HTML5
- CSS3 (avec animations et gradients)
- JavaScript (ES6+)
- Chart.js pour les graphiques

## Thème
Le design évoque l'univers des drones de surveillance avec :
- Couleurs technologiques (bleu cyan, vert)
- Fond étoilé animé
- Effets de glow et transparence
- Animation du logo drone
